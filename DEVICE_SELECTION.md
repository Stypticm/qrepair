# 📱 Выбор устройств для парсинга

## 🎯 Как работает выбор устройств

### **1. Загрузка всех устройств**
- Система загружает **все модели** из базы данных
- Для каждой модели берется **до 5 вариантов** (разные цвета, память, варианты)
- Показывается общее количество доступных устройств

### **2. Фильтрация по моделям**
- **"Все модели"** - показывает все устройства
- **"iPhone X"** - показывает только iPhone X
- **"iPhone 13"** - показывает только iPhone 13
- И так далее для каждой модели

### **3. Выбор устройств**
- **Клик по карточке** - выбирает/отменяет выбор устройства
- **"Выбрать все"** - выбирает все видимые устройства
- **"Очистить"** - отменяет весь выбор

## 🔧 Как использовать

### **Шаг 1: Выберите модель**
1. Откройте страницу "Парсинг цен"
2. В разделе "Выберите устройства" нажмите на нужную модель
3. Например: "iPhone 13 (15)" - покажет 15 вариантов iPhone 13

### **Шаг 2: Выберите конкретные устройства**
1. Кликайте по карточкам устройств для выбора
2. Выбранные устройства подсвечиваются синим
3. Счетчик показывает "X из Y выбрано"

### **Шаг 3: Запустите парсинг**
1. **"Парсить выбранные"** - парсит только выбранные устройства
2. **"Парсить все устройства"** - парсит все видимые устройства

## 📊 Что парсится

### **Для каждого устройства:**
- ✅ **Avito** - б/у устройства (дешевле)
- ✅ **Youla** - частные объявления
- ✅ **Wildberries** - новые устройства (дороже)

### **Результат парсинга:**
- **Средняя рыночная цена**
- **Сравнение с вашей ценой**
- **Статус**: дорого/дешево/нормально
- **Количество найденных цен**

## 🚀 Примеры использования

### **Парсинг одной модели:**
1. Выберите "iPhone 13 (15)"
2. Выберите нужные варианты (например, только 128GB)
3. Нажмите "Парсить выбранные"

### **Парсинг всех устройств:**
1. Выберите "Все модели (45)"
2. Нажмите "Парсить все устройства"

### **Сравнение моделей:**
1. Сначала выберите "iPhone 13" и запустите парсинг
2. Затем выберите "iPhone 14" и запустите парсинг
3. Сравните результаты в разделе "Результаты парсинга"

## ⚡ Оптимизация

### **Рекомендации:**
- **Начните с 1-2 моделей** для тестирования
- **Парсите по 5-10 устройств** за раз
- **Используйте фильтры** для фокуса на конкретных моделях
- **Проверяйте результаты** перед массовым парсингом

### **Производительность:**
- Парсинг 1 устройства ≈ 3-5 секунд
- Парсинг 10 устройств ≈ 30-50 секунд
- Парсинг 50 устройств ≈ 3-5 минут

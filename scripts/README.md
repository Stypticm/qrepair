# Скрипты для работы с ценами устройств

## Описание

Эти скрипты позволяют экспортировать данные о ценах устройств из TypeScript файла в CSV формат для редактирования в Excel, а затем импортировать изменения обратно.

## Файлы

- `export-to-csv.js` - Экспортирует данные из `src/core/appleModels.ts` в CSV файл
- `import-from-csv.js` - Импортирует данные из CSV файла обратно в TypeScript файл
- `appleModels.csv` - CSV файл с данными о ценах устройств

## Использование

### 1. Экспорт в CSV

```bash
node scripts/export-to-csv.js
```

Это создаст файл `scripts/appleModels.csv` с данными о всех устройствах и их ценах.

### 2. Редактирование в Excel

1. Откройте файл `scripts/appleModels.csv` в Excel или другом редакторе таблиц
2. Отредактируйте цены в колонке `basePrice`
3. Сохраните файл в том же формате CSV

### 3. Импорт обратно в TypeScript

```bash
node scripts/import-from-csv.js
```

Это обновит файл `src/core/appleModels.ts` с новыми ценами из CSV файла.

## Структура CSV файла

CSV файл содержит следующие колонки:
- `model` - Модель устройства (например, "X", "11", "12")
- `variant` - Вариант устройства (например, "R", "S")
- `storage` - Объем памяти (например, "64GB", "128GB")
- `color` - Цвет устройства (например, "Bl", "Wh")
- `country` - Страна происхождения (например, "Китай 🇨🇳", "США 🇺🇸")
- `simType` - Тип SIM-карты (например, "1 SIM", "2 SIM", "eSIM")
- `basePrice` - Базовая цена в рублях

## Важные замечания

- Всегда делайте резервную копию файла `appleModels.ts` перед импортом
- Не изменяйте структуру CSV файла (не добавляйте/удаляйте колонки)
- Убедитесь, что цены указаны в рублях и являются целыми числами
- После импорта проверьте, что приложение работает корректно

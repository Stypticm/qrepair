# RotatingBanner Component

## Описание
iPhone-адаптивный компонент для плавной ротации баннеров с оптимизацией для Telegram Mini App. **Декоративный элемент без интерактивности.**

## Особенности
- **iPhone-адаптивный дизайн**: автоматически подстраивается под размеры iPhone экранов
- **Плавные переходы**: использует Framer Motion для качественных анимаций
- **Оптимизация производительности**: предзагрузка изображений, пауза при наведении
- **Telegram WebApp оптимизация**: учитывает контекст Telegram Mini App
- **Мемоизация**: предотвращает избыточные перерендеры

## Использование

```tsx
import { RotatingBanner } from '@/components/RotatingBanner';

<RotatingBanner
  banners={['banner.png', 'banner2.png']}
  interval={5000} // 5 секунд между сменами
/>
```

## Параметры
- `banners`: массив путей к изображениям баннеров
- `interval`: интервал смены в миллисекундах (по умолчанию 4000)
- `className`: дополнительные CSS классы

## iPhone-адаптивные размеры
- iPhone SE (≤375px): 260×55px
- iPhone 12/13/14/15 (≤390px): 280×60px  
- iPhone Pro Max (≤428px): 300×65px
- Desktop (>428px): 320×70px

## Оптимизации
- Предзагрузка всех изображений для плавности
- Пауза анимации при наведении мыши
- Остановка при потере фокуса страницы
- Приоритетная загрузка первого баннера
- Оптимальное качество изображений (85%)

# 🎠 Карусель с Expandable карточками в стиле Apple

## ✅ Что создано:

### 1. Хук `useOutsideClick` (`src/hooks/use-outside-click.ts`)
- Обработка кликов вне компонента
- Поддержка touch и mouse событий
- Автоматическая очистка слушателей

### 2. Компонент `ExpandableDeviceCard` (`src/components/ExpandableDeviceCard.tsx`)
- **Стиль Apple**: Чистый дизайн, скруглённые углы, тени
- **Цвета**: Основной цвет `#2dc2c6` (ваш фирменный)
- **Анимации**: Плавные переходы с Framer Motion
- **Функции**: 
  - Клик для расширения карточки
  - Кнопки: Купить, Избранное, Поделиться
  - Детальная информация в развёрнутом виде
  - Закрытие по Escape или клику вне карточки

### 3. Компонент `DeviceCarousel` (`src/components/DeviceCarousel.tsx`)
- **Карусель в стиле evaluation**: Автопрокрутка, индикаторы
- **Показ по 6 карточек**: 2 ряда по 3 карточки
- **Навигация**: Стрелки влево/вправо, точки-индикаторы
- **Автопрокрутка**: Каждые 4 секунды
- **Загрузка**: Поддержка пагинации и infinite scroll

### 4. Обновлённая главная страница (`src/app/page.tsx`)
- **Интеграция**: Заменили старый `MarketplaceFeed` на `DeviceCarousel`
- **Состояние**: Добавили управление marketplace данными
- **Меню**: Расширили меню новыми пунктами:
  - 📱 Мои устройства
  - 🛒 Маркетплейс  
  - ❤️ Избранное
  - 👤 Профиль
  - ❓ Помощь

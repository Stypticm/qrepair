## 🎯 Как тестировать:

### **Сценарий 1: Карусель**
1. Запустите приложение
2. Увидите 12 товаров в карусели
3. Нажмите на карточку → откроется expandable вид
4. Используйте стрелки для навигации

### **Сценарий 2: Автопереключение**
1. Нажмите "+12 товаров" → станет 24 товара
2. Система автоматически переключится на сетку
3. Протестируйте поиск и фильтры
4. Добавьте ещё товаров для тестирования

### **Сценарий 3: Поиск**
1. Откройте фильтры (кнопка с иконкой фильтра)
2. Введите "iPhone" в поиск
3. Попробуйте сортировку по цене
4. Проверьте счётчик результатов
